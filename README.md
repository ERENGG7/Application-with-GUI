## Description
This project demonstrates a login/registration system with ImGui and C++ and includes extra UI components:

- **Login / Registration**: Secure user authentication
- **Calculator**: Simple arithmetic operations
- **Settings Window**: Change preferences (theme, options)
- **Child Settings Window**: Nested window for advanced settings
